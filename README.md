# Market Quote Assistant

Market Quote Assistant is a Python project designed to support grocery buying decisions by collecting, normalizing, comparing, and recommending product offers across different markets.

## Status

Project is currently under initial structuring and MVP definition.

## Current milestone

Milestone 1 — Project Skeleton

## Main goals

- Compare product prices across markets
- Normalize equivalent products
- Calculate buying scenarios
- Recommend the best purchase strategy

## Initial scope

The MVP will focus on:

- structured shopping list input
- mock market data
- deterministic product normalization
- rule-based comparison
- CLI and JSON outputs

## Tech stack

- Python
- JSON
- modular OOP architecture
